# Obsidian Plugins Workspace

This repository houses all custom-developed and community plugins loaded in the `mainBrain` vault.

## Strategy: Risk Minimization & Self-Maintenance
To prevent unexpected disruptions, breaking updates, or potential security vulnerabilities, this workspace follows a **self-maintenance** strategy:
1. **Fork Community Plugins**: Every community plugin is targeted to be forked to a personal GitHub account.
2. **Self-Host**: Local installations of these plugins will track our personal forks rather than direct community upstream releases.
3. **Local Control**: Any upstream updates will be reviewed, merged, and built manually before deployment.
